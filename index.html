SafePulse — Women Safety Wearable IoT Device

Aug 2025 – Sep 2025

An IoT wearable that instantly sends a real-time Google Maps location link via SMS on SOS button press. Critically, it works on 2G GSM only — no smartphone or internet required.

  • SOS response time under 2 seconds; SMS delivery in 8–12 seconds; GPS accuracy 2.5–5m
  • Interrupt-driven embedded C++ firmware with a 10-second cancel window
  • Resolved SIM800L voltage instability (2A peak spikes) using AMS1117 regulator + 1000µF capacitor
  • Two firmware variants: Arduino+GSM SMS version and ESP32+Bluetooth web-dashboard version

Vision-Based Gesture Controlled Robotic Arm

Jan 2026 – Apr 2026

A real-time, completely contactless robotic arm controlled entirely by hand gestures — no physical controller required. Uses MediaPipe hand landmark detection fed through OpenCV.

  • Mapped 7 distinct hand gestures to 5 servo motor actions: gripper, wrist, elbow, shoulder, base
  • ~100ms gesture recognition latency with ~90% detection accuracy under good lighting
  • 3-frame gesture confirmation buffer eliminates false triggers from rapid hand movements
  • Python-to-Arduino serial communication with batched command packets for synchronisation
